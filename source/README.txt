Setup and run the Futoshiki project:

1. Install Python 3.8 or later.
2. Open a terminal in the project folder:
   cd d:\Project\Futoshiki
3. (Optional) Create and activate a virtual environment:
   python -m venv venv
   .\venv\Scripts\activate
4. Install dependencies:
   pip install -r requirements.txt
5. Run the program:
   python Source\main.py
6. Default input files are in Source\inputs.
7. Output appears in the terminal.

Notes:
- The project uses only Python standard library modules.
- To test with a new file, copy it into Source\inputs and rerun main.py.
